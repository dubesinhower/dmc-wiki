## Mods

### GC Lid Switch PCB

A replacement PCB for the GameCube disc cover lid detection switch. Includes an optional DIP switch to bypass lid detection entirely.

- **Compatible hardware:** Nintendo GameCube
- **Required parts:** KFC-V-105C detector switch, right-angle 3-pos 1.50mm connector header, 1.6mm PCB (standard defaults); optional 1-pos 2.54mm DIP switch for bypass
- **Difficulty:** Beginner
- **Source:** [https://github.com/GarrettBBoyd/GCLidSwitch](https://github.com/GarrettBBoyd/GCLidSwitch)
