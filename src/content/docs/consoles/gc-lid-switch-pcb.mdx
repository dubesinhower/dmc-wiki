## Overview

GCLidSwitch is a custom PCB designed to replace the disc cover lid detection switch in a GameCube. It uses a KFC-V-105C detector switch and connects via a right-angle 3-position 1.50mm header. An optional single-position DIP switch allows the lid detection to be bypassed entirely.

PCB thickness is 1.6mm (standard default).

## Compatible Hardware

- Nintendo GameCube

## Notes

- The optional DIP switch bypasses lid detection, useful if the original switch is unreliable or for open-lid use cases.
- Order the PCB at 1.6mm thickness with default settings.
